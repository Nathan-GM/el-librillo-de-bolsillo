# Introducción
Este apartado muestra una breve introducción sobre que es es este proyecto.

## ¿Qué es?
![Logo de la aplicacion](imgs/logo.png)

El librillo de bolsillo es una aplicación que permite la compra de productos organizados
en distintos géneros que cree los administradores.

## ¿Cuál es su objetivo?
El objetivo de este proyecto es ofrecer una aplicación que permita a usuarios
comprar diferentes libros y una manera sencilla a los encargados de gestionar la
aplicación.

## ¿Que problemas se quiere resolver?
Con esta aplicación se quiere hacer 3 cosas:
1. Dar facilidades a los encargados para gestionar las aplicaciones, en este caso llevar la tienda.
2. Darles una forma de interactuar con la base de datos sencilla, sin necesidad de tener conocimientos.
3. Ofrecer a los usuarios un lugar donde encontrar articulos sea sencillo.
