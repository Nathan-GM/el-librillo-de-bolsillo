# Roles.
- [No registrado](#no-registrado)
- [Usuario](#usuario-registrado)
- [Administrador o Admin](#administrador-o-admin)

Para su correcto funcionamiento, los usuarios se separan en distintos roles que les permiten hacer distintas acciones.

Estos roles son:
- No registrado
- Usuario
- Administrador / Admin.

El primero, pese a no ser un rol como tal, es importante saber que existe, ya que para poder realizar compras, reseñas o ver carritos se ha de usar una cuenta, pero para ver los articulos que existen no es necesario crearla. Así pues, se explicará lo que puede y no puede hacer cada usuario.

### No registrado
Estos son los usuarios que no han creado cuentas. Estos pueden:
- Ver los productos existentes en la página
- Ver las reseñas de otros usuarios.

Mientras que si quieren realizar compras o publicar sus propias reseñas se les solicitará crearse una cuenta.

### Usuario Registrado
Esto representa a aquellos usuarios que si tienen cuenta. Estos cuentan con las siguientes funcionalidades:
- Beneficios de los usuarios no registrados.
- Capacidad de comprar productos.
- Capacidad de hacer sus propias reseñas.
- Posibilidad de ver antiguos carritos (compras).

### Administrador o Admin
Estos son los usuarios encargados de administrar la tienda (gerentes, encargados, dueños, etc). A este tipo de usuario se le permite lo siguiente:

- Beneficios de los usuarios registrados.
- Capacidad de listar todos los productos, viendo su stock y si estos se han eliminado.
- Capacidad de agregar, editar y eliminar nuevos productos.
- Capacidad de crear nuevos géneros
- Capacidad de eliminar reseñas.
- Capacidad de convertir usuarios existentes en administradores y viceversa.

## Permisos de No Registrados:

- Ver productos
- Ver reseñas

## Permisos de Usuario:

- Permisos de no registrados
- Comprar articulos
- Hacer reseña
- Ver antiguos productos

## Permisos de administradores

- Permisos de usuario
- Ver todos los productos, incluso eliminados
- Crear nuevos productos
- Editar productos existentes
- Borrar productos existentes 
- Crear nuevos géneros de productos
- Eliminar reseñas
- Convertir usuarios en admin
- Convertir administradores en usuarios
